Respuestas del cuestionario

  1. Dar un par de ejemplos de uso de etiquetas meta: para definir el autor, el charset o para agregar palabras clave para motores de busqueda
  2. Nombrar al menos 2 atributos aplicables a la etiqueta HTML: xmlns, lang, translate, id, class, style, data-*, hidden, tabindex
  3. Como se relaciona un header http con la etiqueta head de HTML? La etiqueta head contiene metadatos del documento que son utilizados en el header http
  4. Que es UTF8? UTF8 es la codificacion o set de caracteres utilizados en el documento
  5. Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable? El atributo para aplicar estilo es style y se puede aplicar a cualquier etiqueta ya que es un atributo global
  6. Donde se indica que un enlace debe ubicar el documento en una nueva pestaña? Se indica con el atributo target y el valor _blank
  7. Explicar cuando usarían etiquetas semánticas y con que propósito Usaria etiquetas semanticas para estructurar el documento y para que se indexe a los motores de busqueda
  8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar Usaria border:solid para mostrar bordes solidos y border-collapse:collapse para que los bordes de celdas contiguas colapsen en uno solo
  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP? La etiqueta body es donde va el contenido de un documento HTML, mientras que el body del protocolo contiene los datos de respuesta de una request
  10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican? Los atributos colSpan y rowSpan marcan la agrupacion de celdas, estos pueden ser usados en etiquetas td
